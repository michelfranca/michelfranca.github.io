### Projeto 1: Análise de Vendas Globais
* **Status:** ⏳ Em desenvolvimento
* **Descrição:** Um dashboard interativo para analisar o desempenho de vendas de uma rede de varejo fictícia, permitindo a exploração de resultados por região, produto e tempo.
* **Objetivos de Aprendizagem:**
  - Praticar modelagem Star Schema.
  - Desenvolver medidas DAX de Time Intelligence (YTD, MTD).
  - Implementar Row-Level Security (RLS) para gerentes regionais.
* **Ferramentas:** Power BI, SQL.

### Projeto 2: Análise de Recursos Humanos (RH)
* **Status:** 📋 Planejado
* **Descrição:** Um painel para o departamento de RH, focado em métricas como taxa de rotatividade (turnover), tempo de contratação, satisfação dos funcionários e distribuição demográfica.
* **Objetivos de Aprendizagem:**
  - Calcular métricas complexas de RH com DAX.
  - Utilizar visuais para análise de tendências e distribuições.
  - Praticar a limpeza e transformação de dados de múltiplas fontes (planilhas, sistemas de RH).
* **Ferramentas:** Power BI, Excel.
