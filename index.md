# Bem-vindo ao Hub de Conhecimento!

Esta é a minha plataforma centralizada para documentar estudos, construir guias práticos e apresentar projetos de dados.

O objetivo é criar uma base de conhecimento de alta qualidade para a comunidade, começando com **SQL** e **Power BI**.

Navegue pelas seções no menu à esquerda para explorar todo o conteúdo.
