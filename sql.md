# Roadmap de Estudos: SQL

[Voltar para a Página Inicial](index.md )

---

Este é o plano de estudos completo para SQL, dividido por níveis de experiência. Cada tópico será um link para uma página com explicações, exemplos e exercícios.

## Nível Júnior: Os Fundamentos

O objetivo aqui é ser capaz de fazer consultas simples e extrair dados de uma ou mais tabelas.

- [ ] **Fundamentos de Consulta:** `SELECT`, `FROM`, `WHERE`
- [ ] **Ordenação e Limite:** `ORDER BY`, `LIMIT`
- [ ] **Agregações Básicas:** `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- [ ] **Agrupamento de Dados:** `GROUP BY`, `HAVING`
- [ ] **Manipulação de Dados:** `INSERT`, `UPDATE`, `DELETE`

## Nível Pleno: Consultas Complexas

O foco é combinar e transformar dados de maneiras mais avançadas.

- [ ] **Junção de Tabelas (Joins):** `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL OUTER JOIN`
- [ ] **Subqueries (Subconsultas):** Usando uma consulta dentro de outra.
- [ ] **Common Table Expressions (CTEs):** A cláusula `WITH` para organizar e reutilizar lógicas.
- [ ] **Tratamento de Dados:** `CASE WHEN`, `COALESCE`, `NULLIF`

## Nível Sênior: Otimização e Tópicos Avançados

Aqui, o foco muda para performance, escalabilidade e automação.

- [ ] **Funções de Janela (Window Functions):** `ROW_NUMBER()`, `RANK()`, `LEAD()`, `LAG()`
- [ ] **Performance de Queries:** Entendendo Índices e `EXPLAIN PLAN`.
- [ ] **Controle de Transações:** `BEGIN TRANSACTION`, `COMMIT`, `ROLLBACK`.

## Nível Especialista: Arquitetura e Modelagem

- [ ] **Modelagem de Banco de Dados:** Formas Normais (1FN, 2FN, 3FN), Star Schema.
- [ ] **Tópicos Avançados:** Stored Procedures, Triggers, Views.
